Support

For support, inspect recent writes in the dashboard, use undo for the latest ingestion run, or remove profile-context records from the Profile tab.

For deletion requests, identify the memory or profile-context record in the dashboard and remove it, or use Brain's authenticated MCP tools to forget the record.

Operators can verify deployment readiness with make prod-check and make cloudflare-verify. These checks include the public dashboard, support pages, OAuth metadata, curated MCP surface, app component resource, and security headers.
