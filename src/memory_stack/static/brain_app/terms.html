Brain

Terms

Brain is a private personal-memory service. Use it only with data you are allowed to store and retrieve.

Read operations recall stored memory. Write operations on the public app surface preview first and require explicit confirmation before storage or deletion.

Administrative projection, rebuild, raw Cognee, hard-delete, and system user-management operations are not available on the public app surface.

The dashboard exposes user-level controls for reviewing memories, profile context, account password changes, and recent write activity. System administration requires a superuser account.
